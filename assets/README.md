Pasta `assets/` — imagens e arquivos estáticos do projeto

Coloque aqui o arquivo do logotipo como:

  assets/imbel-logo.png

Recomendações:
- Formato: PNG com fundo transparente (preferível).
- Altura ideal: ~56px para encaixar no cabeçalho (o CSS é responsivo).
- Nome sugerido: `imbel-logo.png` (o HTML já referencia esse nome).

Como adicionar o arquivo:
- Abra o Explorer do VS Code e arraste o arquivo para esta pasta, ou
- Responda nesta conversa com o arquivo de imagem (eu posso salvá-lo aqui), ou
- Você pode sobrescrever via terminal/local copy.

Se quiser, posso salvar o anexo desta conversa como `assets/imbel-logo.png` agora — confirme "sim".
